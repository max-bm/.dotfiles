# .dotfiles

A repository for storing my personal dotfiles

## Instructions

Clone the repository, change directory, and run the install script:

    ```bash
    git clone git@github.com:max-bm/.dotfiles.git
    cd .dotfiles
    ./setup.sh
    ```
